Internet A global computer network providing a variety of information and communication facilities.
HyperText Transfer Protocol HTTP The communications protocol used to connect to web servers on the internet or on a local interwork.
HyperText Markup Language HTML A standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on www pages.
World Wide Web WWW All public websites or pages that users can access on their local computers.
World Wide Web Consortium W3C An international community that works together for the long term growth of the Web.
Americans with Disabilities Act ADA An act passed in 1990 that creates a standard to make websites more accessible for people with disabilities.
The World Intellectual Property Organization WIPO An international organization dedicated to promoting the use and protection of works of the human spirit.
Client A computer of workstation that is capable of obtaining information and applications from a server.
Server A computer or program which manages access to a centralized resource or service in a network.
Client/Server Denoting a computer system in which a central server provides data to a number of networked workstations.
Multipurpose Internet Mail Extensions MIME An extension of the original SImple Mail Transport Protocol. It lets users exchange different kinds of data files.
Simple Mail Transfer Protocol SMTP An internet standard communication protocol for electronic mail transmission.
Post Office Protocol 3 POP3 The most commonly used protocol for receiving email over the internet. It's sued to receive emails from a remote server and send to a local client.
Internet Message Access Protocol IMAP It manages and retrieves email messages from a receiving server.
File Transfer Protocol FTP A way to download, upload, and transfer files from one location to another on the internet or between computers.
Transmission Control Protocol TCP An important network protocol that lets two hosts connect and exchange data streams.
Internet Protocol IP A unique label assigned to websites and servers as well as digital devices.
Domain Name System DNS Translates human readable domain names to machine readable IP Addresses
Uniform Resource Locator URL A URL is the address of a giving resource on the Web.
Domain Name The name of a website.
Fully Qualified Domain Name An complete domain name for a specific computer or host on the internet.
Top-Level Domain TLD The most generic domain name in the DNS hierarchial system. EX. .org
Markup Language A text-encoding system consisting of a set of symbols inserted in a text document to control it's structure and formatting.
Element A part of a webpage, in html it may contain a data item or chuck of text or image. It could also contain nothing.
Tag A piece of information that describes the data or content that it is assigned to.
Attribute A characteristic of a page element, such as font size or color.
Extensible HyperText Markup Language XHTML A stricter, more XML-based version of html.
HyperText Markup Language 5 HTML5 It is the current version of HTML5 with new features to make it easier.
Document Type Definition DTD Defines the structure and the legal elements and attributes of an XML document.
DOCTYPE The HTML declaration. It tells the browsers about what version of HTML the page is written in.
